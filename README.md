# 🗽 10 Most Famous Places in New York City

An immersive and cinematic web experience showcasing the **10 most famous landmarks in New York City** through stunning visuals, interactive storytelling, animated transitions, historical insights, and modern web design.

---

## ✨ Features

### 🎬 Cinematic Landing Page

- Full-screen hero section
- Smooth entrance animations
- Dynamic New York City atmosphere
- Mobile-first responsive design

### 🗽 10 Iconic NYC Landmarks

Explore detailed information about:

1. Statue of Liberty
2. Times Square
3. Central Park
4. Empire State Building
5. Brooklyn Bridge
6. Rockefeller Center
7. Grand Central Terminal
8. Fifth Avenue
9. One World Trade Center
10. Broadway

### 📖 Storytelling Experience

Each landmark includes:

- Historical background
- Construction timeline
- Cultural significance
- Visitor information
- Interesting facts
- Location details
- Annual visitor statistics

### 🎨 Interactive UI

- Smooth scrolling navigation
- Scroll-based animations
- Framer Motion transitions
- Responsive image backgrounds
- Dynamic chapter layout
- Modern glassmorphism effects

### 📱 Fully Responsive

Optimized for:

- Mobile Phones
- Tablets
- Laptops
- Desktop Monitors
- Ultra-Wide Displays

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/AflahaSetyaningrum/10-Most-Famous-Places-in-New-York-City.git
```

Enter the project folder:

```bash
cd 10-Most-Famous-Places-in-New-York-City
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
